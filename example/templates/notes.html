mustache.couch.js live updating example

Open this page in two browser windows - when you add a note, both will update. Refresh the page, and all notes will be rendered without an Ajax call.

Initial page load and Ajax row updates are both served from the same list function. You can also add a live updating list to a static _attachments page - View example here.
